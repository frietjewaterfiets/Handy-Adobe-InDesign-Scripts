**InDesign Copy Linked File Name**

This Adobe InDesign script copies the filename of the selected linked image directly to the clipboard ‚Äî quickly, silently, and without showing alerts.

This script is super handy when you need to quickly grab image filenames ‚Äî for example, when organizing assets in Excel, file lists, or documentation.
If you plan to use it often, consider assigning it to a keyboard shortcut in InDesign for instant access.

**InDesign Rapid Frame Filler**

These scripts let you instantly resize shapes and image frames to perfectly fill your InDesign pages ‚Äî with bleed or within margins.‚Äù

**Recommended shortcuts:**

Ctrl + Alt + F ‚Üí Margin Filler

Ctrl + Alt + D ‚Üí Full Bleed Filler

üìñ **Usage**
1. Select any shape or image frame on your page.
2. Run one of the scripts from the Scripts panel (or use your shortcut).
3. The frame will automatically resize:
4. Margin Filler ‚Üí fits neatly within your page margins.
5. Full Filler ‚Üí expands to cover the entire page + bleed.
6. If the frame contains an image, the content will scale proportionally to fill the frame.

These scripts only auto-fit images placed inside rectangular frames.
Placed PDFs  will not be resized automatically ‚Äî the frame will resize, but the content will not scale.
